# Data Dictionary

This document describes every table used in the project. Hebrew column names are kept as-is
(they are the official CBS / NII labels); descriptions below explain their meaning.

---

## 1. Raw sources (`data/raw/`)

### 1.1 `Election_data/Knesset_XX_Election_Results_*.csv` (5 files, Knesset 21–25)

One row per locality. Source: Israeli Central Elections Committee. Encoding varies (UTF-8 / Windows-1255, handled in code).

| Column(s) | Type | Meaning |
|---|---|---|
| סמל ועדה | int | Elections-committee code (present in some files; unused) |
| שם ישוב | text | Locality name |
| סמל ישוב | int | Official locality code — the merge key |
| בזב | int | Eligible voters ("baalei zchut bchira") |
| מצביעים | int | Total votes cast |
| פסולים | int | Invalid votes |
| כשרים | int | Valid votes — the denominator for all shares |
| ~35–45 single/multi-letter columns (e.g., מחל, פה, שס...) | int | Votes for each party ballot letter. The letter-to-party assignment changes between elections; the mapping used is documented in notebook 01 |

### 1.2 `Education_data/Education_YYYY.xlsx` (2019–2023)

One row per locality, ~39 indicators per year. Source: CBS. Column groups:
kindergarten enrollment by age (ילדים בגנים...), schools and classes by level (בתי ספר..., כיתות...),
class sizes (ממוצע תלמידים לכיתה...), matriculation rates (אחוז זכאים לתעודת בגרות...,
including the university-threshold variant), teaching-staff indicators (עובדי הוראה... — hours,
students per teacher, share with advanced degrees), dropout rates (אחוז תלמידים נושרים...),
and higher-education shares (השכלה גבוהה... — students among ages 20–25, degree holders among 35–55).

### 1.3 `Demographic_data/Demographic_YYYY.xlsx` (2019–2023)

One row per locality, ~66 indicators per year. Source: CBS. Column groups:
total population (סה"כ אוכלוסייה בסוף השנה), population density (צפיפות אוכלוסייה...),
population-group shares (יהודים ואחרים %, ערבים %, מוסלמים %, נוצרים %, דרוזים % — the last
three measured out of the Arab population), age-structure shares (אחוז באוכלוסייה בסוף השנה בני X-Y),
immigration (אחוז עולי 1990+ מסך האוכלוסייה, השתקעות ראשונה של עולים...), vital statistics
(ריבוי טבעי ל-1,000, שיעור פריון כולל, נישאים/מתגרשים ל-1,000), internal migration
(נכנסים/יוצאים מיישובים אחרים לפי גיל), dependency ratio (יחס תלות), and institutional-housing
shares (אוכלוסיית דיור משותף... — mostly missing, filtered out).

### 1.4 `Average_income/ICBS_YYYY.xlsx` (2019–2023)

One row per locality. Source: CBS. **The schema changes across years** (handled in code):

| Measure | 2019 | 2020 | 2021–2023 |
|---|---|---|---|
| Average monthly wage (total / men / women) | שכר ממוצע לחודש של שכירים (ש"ח)... | same | כלל בעלי הכנסה שכר ממוצע / גברים / נשים שכר ממוצע |
| Share earning up to minimum wage | אחוז השכירים המשתכרים עד שכר מינימום | — | derived: שכירים השתכרים מתחת לשכר המינימום מספר ÷ בעלי הכנסה משכר מספר |
| Wage Gini index | מדד אי-השוויון שכירים (מדד ג'יני) | — | — |

### 1.5 `Unemployment_data/Unemployment_YYYY.xlsx` (2019–2023)

One row per locality, ~29 indicators per year. Source: National Insurance Institute. Column groups:
unemployment-benefit recipients (מספר/אחוז מקבלי דמי אבטלה..., by gender, average age, benefit
amounts and utilization days), old-age and survivors pensions (מקבלי קצבאות זקנה ושאירים...,
income-supplement share), income-support recipients (מקבלי גמלת הבטחת הכנסה), child allowances
by family size (מספר ילדים שבגינם שולמו קצבאות...), disability benefits, and HMO membership shares
(אחוז המבוטחים בקופ"ח... per HMO).

### 1.6 `Locality_Types.xlsx`

Static classification, one row per locality (1,270 rows).

| Column | Meaning |
|---|---|
| שם יישוב | Locality name (Latin script) |
| סמל יישוב | Official locality code — merge key |
| צורת יישוב | CBS settlement-form numeric code |
| type | Category used by the project: `Cities`, `Moshavim`, `Kibbutzim`, `Arab/Non-Jewish`, `other` |
| סוג יישוב | Hebrew description of the size/type class |

## 2. Processed tables (`data/processed/`) — generated by the notebooks

### 2.1 `merged_elections_demographics.csv` — output of notebook 01

1,124 rows (locality × election) × ~154 columns: merge keys (`locality_symbol`, `שם יישוב`, `year`,
`target_election`), all harmonized source indicators (cleaned of year labels), the income measures
(`avg_monthly_wage`, `pct_below_minimum_wage`, `gender_wage_ratio`, `gini_wage_2019`), and the
election-derived columns listed in 2.2.

### 2.2 `election_targets_by_locality.csv` — output of notebook 01

6,073 rows — **all** ~1,215 localities × 5 elections (not only the 226 modeled ones).

| Column | Meaning |
|---|---|
| `locality_symbol`, `election_locality_name`, `target_election`, `year` | Identifiers |
| `valid_votes` | Valid votes (כשרים) |
| `Right`, `Center_Left`, `Haredi`, `Arab` | Raw vote counts per bloc |
| `Other` | Valid votes not mapped to any bloc |
| `total_main_blocks` | Sum of the four bloc counts — the denominator of the targets |
| `Right_pct`, `Center_Left_pct`, `Haredi_pct`, `Arab_pct` | **Targets**: bloc share of the four-bloc total, sums to 100 |
| `Other_raw_pct` | Other as % of valid votes (audit only) |

### 2.3 `modeling_dataset_selected_features.csv` — output of notebook 02

1,124 rows × 44 columns. The model's input table.

| Group | Columns | Notes |
|---|---|---|
| Identifiers (4) | `locality_symbol`, `שם יישוב`, `year`, `target_election` | Never used as features |
| Audit (2) | `valid_votes`, `Other_raw_pct` | Used for vote-weighted evaluation only |
| Targets (4) | `Right_pct`, `Center_Left_pct`, `Haredi_pct`, `Arab_pct` | Sum to 100 per row |
| Selected features (28) | See list below | May contain NaN by design (imputed inside the model pipelines) |
| Locality type (6) | `type_Arab/Non-Jewish`, `type_Cities`, `type_Kibbutzim`, `type_Moshavim`, `type_other`, `type_Druze_Majority` | One-hot + engineered flag; `type_Arab/Non-Jewish` is the segmentation routing variable |

The 28 selected demographic features, by theme:

- **Population composition (7):** אחוז עולי 1990+ מסך האוכלוסייה · ערבים (אחוזים מתוך כלל אוכלוסיית הישראלים) · יהודים ואחרים (אחוזים...) · יהודים (אחוזים מתוך יהודים ואחרים) · מוסלמים (אחוזים מתוך האוכלוסייה הערבית) · צפיפות אוכלוסייה לקמ"ר... · נכנסים מיישובים אחרים גיל 30-64
- **Age and family (6):** אחוז בני 0-4 · אחוז בני 30-44 · אחוז בני 45-59 · ריבוי טבעי ל-1,000 · נישאים שיעור ל-1,000... · יחס תלות
- **Education (7):** אחוז זכאים לתעודת בגרות · אחוז זכאים... בדרישות הסף של האוניברסיטאות · 3 מדדי עובדי הוראה (שעות לתלמיד, שעות לשבוע, אחוז תואר שני+) · אחוז סטודנטים מבני 20-25 · אחוז בעלי תארים מבני 35-55
- **Economy and wages (4):** `avg_monthly_wage` · `gini_wage_2019` (static, 2019 measurement) · `gender_wage_ratio` (male/female average wage) · מקבלי דמי אבטלה (ממוצע חודשי) גברים
- **Welfare (4):** מספר ילדים... במשפחות עם 5 ילדים ויותר ל-1,000 תושבים (normalized) · אחוז מקבלי השלמת הכנסה מבין מקבלי קצבאות זקנה · אחוז המבוטחים בקופ"ח כללית · גיל ממוצע של מקבלי דמי אבטלה

### 2.4 `selected_demographic_features.json`

The exact selected-feature list plus the selection metadata (method, thresholds, seed).

### 2.5 `validation_predictions_selected_model.csv` — output of notebook 03

226 rows — the Knesset-25 **test** predictions of the final model: identifiers, locality-type flags,
actual targets, `Predicted_*`, `Absolute_Error_*`, `Mean_Absolute_Error`, `Predicted_Total`.

### 2.6 `coverage_by_election.csv`

Vote coverage of the modeled dataset per election (original vs. merged valid votes).

## 3. Reports (`reports/`)

| File | Content |
|---|---|
| `model_comparison.csv` / `model_comparison_by_target.csv` | Selection-phase results (Knesset 24) for all nine architectures, overall and per target |
| `modeling_summary.json` | Selected model, selection and test metrics, feature counts |
| `evaluation_summary.csv` / `.json` | Final test metrics recalculated independently in notebook 04 |
| `error_analysis_by_locality.csv` | Per-locality test errors, signed and absolute, with analysis groups |
| `error_summary_by_group.csv` / `error_summary_by_group_and_target.csv` | Error aggregates by locality group (and bloc) |
| `feature_importance_by_target.csv` | Full Random-Forest importance ranking per target |
| `training_missingness_summary.csv` | Missing-rate audit per feature |
| `preprocessing_summary.json` | Preprocessing configuration and counts |
| `figures/` | All charts generated by notebooks 02 and 04 |
