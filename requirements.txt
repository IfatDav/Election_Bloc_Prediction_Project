### Recommended Environment

- Python 3.10 or later
- Google Colab or Jupyter Notebook
- Google Drive when running the current Colab-based workflow

### Clone the Repository

```bash
git clone https://github.com/IfatDav/Election_Prediction_Project.git
cd Election_Prediction_Project
```

### Create a Virtual Environment

Linux or macOS:

```bash
python -m venv .venv
source .venv/bin/activate
```

Windows:

```bash
python -m venv .venv
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Google Colab Directory

The current notebooks use the following default Google Drive directory:

```text
/content/drive/MyDrive/לימודים/פרויקט DS
```

When using another path, update the following variable near the beginning of each notebook:

```python
PROJECT_DIR = Path("/your/project/path")
```

---

## How to Run the Project

Run the notebooks in the following order:

1. `01_data_mapping_and_merging.ipynb`
2. `02_eda_and_preprocessing.ipynb`
3. `03_modeling.ipynb`
4. `04_model_evaluation.ipynb`

Each notebook should be executed from top to bottom using **Run all**.
